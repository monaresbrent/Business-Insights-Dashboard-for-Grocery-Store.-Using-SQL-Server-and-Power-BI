# Business-Insights-Dashboard-for-Grocery-Store.-Using-SQL-Server-and-Power-BI

This project aims to transform raw sales data into actionable business insights for a Grocery Store. Currently, the business lacks the visibility needed to track key performance metrics, such as top-selling products, high-value customers, and revenue trends.
By centralizing sales data within SQL Server and developing an interactive dashboard in Power BI, this solution will empower the business owner to analyze performance effectively and make informed, data-driven decisions to grow the business.

# Sales Analytics Dashboard

## Project Overview
This project analyzes sales performance using Power BI.

## Tools Used
- Power BI
- SQL Server

## KPIs
- Total Sales
- Total Orders
- Total Customers
- Total Products

## Dashboard Features
- Sales Trend
- Top Products
- Sales by Category
- Regional Sales
- Interactive Filters
